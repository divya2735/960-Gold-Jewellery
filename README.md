A full-stack web-based e-commerce application for browsing and purchasing jewellery online. This project provides a seamless and user-friendly shopping experience with features like cart management, favorites, reviews, and secure payment options.

---

## 🚀 Features

- Browse Jewellery Products (Rings, Necklaces, etc.)
- Search and Filter Products
- Add to Favorites (Wishlist)
- Add to Cart and Manage Cart
- Product Reviews and Ratings
- Payment Options (Cash on Delivery + Online Payment)
- User Authentication (Login/Register)
- Order Management System
- Fully Responsive Design

## ⚙️ Installation & Setup

1. Clone Repository
git clone https://github.com/your-username/jewellery-store-project.git
cd jewellery-store-project

2. Install Dependencies
pip install django

3. Apply Migrations
python manage.py makemigrations
python manage.py migrate

4. Create Superuser
python manage.py createsuperuser

5. Run Server
python manage.py runserver

6. Open in Browser
http://127.0.0.1:8000/
